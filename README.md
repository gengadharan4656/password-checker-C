# 🔐 Password Strength Checker & Generator (C)

A **console-based C application** that allows users to:
- Check the strength of a password
- Generate strong passwords
- Create customized passwords using user-defined keywords

This project is designed for **beginners**, **academic mini-projects**, and **seminar demonstrations**.

---

## 📌 Features

- ✅ Menu-driven console application
- ✅ Password strength analysis:
  - Length check
  - Numeric character detection
  - Special character detection
- ✅ Strength classification:
  - Weak
  - Medium
  - Strong
- ✅ Password generation modes:
  - Fully random password
  - Customized password using user keyword (name)
- ✅ Uses core C concepts only

---

## 🧠 Concepts Used

- Functions

- Conditional statements (`if-else`)

- Random number generation (`rand()`, `srand()`)
